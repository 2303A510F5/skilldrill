## 📖 Overview
**SkillDrill** is a premium, high-performance web application designed to test engineering fundamentals across multiple domains. Built from the ground up using **pure Vanilla Web Technologies**, this project completely eliminates the need for heavy frameworks or backend databases by fully leveraging the power of modern client-side APIs, LocalStorage data persistence, and dynamic DOM manipulation.

## ✨ Key Features
- ⚡ **Framework-Free SPA**: Operates as a seamless Single Page Application utilizing native JavaScript logic instead of React/Node.
- 🎨 **Glassmorphism UI**: A modern, responsive dark-mode aesthetic powered by advanced CSS backdrop filters, flexbox, and gradients.
- 🧠 **Robust Quiz Engine**: 80 expertly curated engineering questions spanning 8 dedicated subjects (Data Structures, Circuits, Thermodynamics, Structural Analysis, etc). Features a Fisher-Yates array shuffling implementation to guarantee random options and questions on every attempt.
- 📊 **Persistent Analytics**: A dedicated Analytics dashboard that records your lifetime submission history, accuracy percentages, and builds dynamic Doughnut Charts using `Chart.js`.
- 🔍 **Detailed Post-Action Review**: Users can review their past quiz attempts to see exactly what they answered, the correct answers, and read detailed technical explanations for every engineering question.

## 🛠️ Technology Stack
* **Structure:** HTML5 Semantic Markup
* **Styling:** CSS3 (Custom Variables, CSS Animations, Glassmorphism, Responsive Grid/Flexbox)
* **Logic:** Vanilla JavaScript (ES6+ Features, DOM manipulation, JSON state management)
* **Storage:** Browser `localStorage` API
* **Data Viz:** Chart.js (CDN)
* **Icons:** FontAwesome 6

## 🚀 Quick Start & Installation

Because SkillDrill is completely self-contained, no complex server setups, package installations (`npm install`), or databases are required! 

### Online Live Demo
You can view the fully functional live deployment of this project hosted on GitHub pages right here:
👉 **[View SkillDrill Live](https://2303A510F5.github.io/skilldrill/)**

### Run Locally
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/2303A510F5/skilldrill.git
   ```
2. Open the folder.
3. Double-click the `index.html` file to instantly run the app in your default web browser.
